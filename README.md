Machine Learning Algorithms: Clustering & ClassificationThis repository demonstrates the implementation and evaluation of fundamental Machine Learning algorithms using Python. The project focuses on unsupervised learning via K-means Clustering and supervised learning using Decision Tree Classifiers, applying them to the classic Iris dataset.🚀 Key Implementations1. K-means Clustering (Unsupervised Learning)Concept: An algorithm that classifies data points based on proximity to centroids and majority voting of neighbors.Workflow:Choosing $k$ clusters and a distance metric.Identifying the $k$ nearest neighbors for classification.Iteratively updating centroids until convergence.Implementation: Uses sklearn.cluster.KMeans to group data without prior labels.2. Decision Tree Classifier (Supervised Learning)Goal: Building a model to predict classes by learning simple decision rules inferred from data features.Optimization:Performed a depth sweep (from 1 to 20) to find the optimal tree depth.Visualized the tradeoff between training and testing scores to identify the "sweet spot" before overfitting occurs.Evaluation: Uses DecisionTreeClassifier from sklearn.tree.📊 Dataset: Iris DatasetThe project utilizes the Iris dataset, a staple in machine learning for pattern recognition.Features: Sepal length, sepal width, petal length, and petal width.Scaling: Note that for distance-based algorithms like K-means, features should be on the same scale (the Iris dataset features are already comparable in scale).🛠️ Requirements & DependenciesTo run this notebook, you will need:Python 3.xscikit-learn: For machine learning model implementation.pandas: For data manipulation and plotting results.matplotlib: For visualizing model performance.Install the dependencies using pip:Bashpip install scikit-learn pandas matplotlib
